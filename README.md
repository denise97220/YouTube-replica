# Youtube Replica

復刻 Youtube 首頁，還原首頁的畫面與功能列，包含影片顯示、滑鼠移入後播放等。

## 產品功能

* 復刻 Youtube 首頁，還原首頁的畫面與功能列。
* 包含影片顯示、滑鼠移入後播放等。
* RWD 響應式網站設計，支援多種裝置介面瀏覽。
